This program is run in the command line by executing 'javac *.java' in the working directory.
Then run 'java AlertSystem' to see the output. The output shows the implementation of a 
NotificationMedium in an AlertSystem, changing from SMS to Email to WhatsApp. Messages are output and 
stored in a log, which is also output. 

Any queries from ChatGPT are in the class documentation.
